<h1 align="center">DataBar</h1>

DataBar is a lightweight macOS menu bar app that displays your Google Analytics real-time user count at a glance.

## Features

- Lives in your menu bar - always visible, never in the way
- Shows real-time active users from Google Analytics
- Click to jump directly to Google Analytics dashboard
- Automatic refresh (configurable interval)
- Sign in with your Google account

## Requirements

- macOS 13.0 or later
